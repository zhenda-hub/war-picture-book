# 历史战争场景

## 古代时期 (公元前3000年 - 公元500年)

### 【卡迭石战役】公元前1274年

- **交战方**: 埃及帝国 vs 赫梯帝国
- **地点**: 叙利亚卡迭石（奥龙特河畔）
- **关键人物**: 拉美西斯二世 vs 穆瓦塔里二世
- **兵力**: 埃及~20,000人 vs 赫梯~37,000人（含战车）
- **兵种**:
  - 埃及: 战车(双人)、弓箭手、步兵、努比亚雇佣兵
  - 赫梯: 三人战车(重装)、步兵
- **地形特征**: 河流、丘陵、古城要塞
- **场景要点**: 战车冲锋、河流地形、古城墙、埃及与赫梯的不同战车样式

### 【希波战争】公元前499-449年

- **交战方**: 希腊城邦 vs 波斯帝国
- **关键战役**:
  - **马拉松战役** (公元前490年)
    - 交战方: 雅典(约1.1万) vs 波斯帝国(约1-2.5万)
    - 关键人物: 米太亚德(雅典) vs 达提斯(波斯)
    - 兵种: 雅典重装步兵( hoplites ) vs 波斯轻步兵/弓箭手/骑兵
    - 地形: 马拉松平原、沼泽地
    - 场景要点: 希腊方阵冲锋、波斯弓箭手阵型、雅典双翼包抄战术
  - 温泉关战役 (公元前480) - 斯巴达300勇士
  - 萨拉米斯海战 (公元前480) - 希腊海军大胜
- **地形**: 海岸、山地峡谷、海洋
- **场景要点**: 希腊方阵、波斯弓箭手、三桨座战船、温泉关峡谷

### 【亚历山大东征】公元前334-323年

- **交战方**: 马其顿帝国 vs 波斯帝国
- **关键战役**:
  - 伊苏斯战役 (公元前333)
  - 高加米拉战役 (公元前331) - 决定性一战
- **兵种**:
  - 马其顿: 方阵步兵、伙友骑兵、投石兵
  - 波斯: 不死军、战车、骑兵、大象
- **场景要点**: 马其顿方阵、大规模战车冲锋、波斯战象

### 【长平之战】公元前260年

- **交战方**: 秦国 vs 赵国
- **关键人物**: 白起(秦) vs 赵括(赵)
- **兵力**: 秦军约40万 vs 赵军约45万
- **兵种**:
  - 秦军: 精锐步兵、骑兵、弩兵
  - 赵军: 征召步兵、骑兵
- **地形**: 山西山地、上党高地
- **场景要点**: 山地围困战、秦军包围工事、坑杀场景(历史争议)

## 古典时代 (公元前500年 - 公元500年)

### 【布匿战争】公元前264-146年

- **交战方**: 罗马共和国 vs 迦太基
- **关键战役**:
  - 坎尼战役 (公元前216) - 汉尼拔的包围战术
  - 扎马战役 (公元前202) - 西庇阿击败汉尼拔
- **兵力**: 罗马 ~50,000+ vs 迦太基 ~40,000+
- **兵种**:
  - 罗马: 军团步兵、努米底亚骑兵、战象
  - 迦太基: 凯尔特/伊比利亚步兵、战象、骑兵
- **场景要点**: 罗马龟甲阵、战象、坎尼半月形阵

### 【高卢战争】公元前58-50年

- **交战方**: 罗马共和国 vs 高卢部落
- **关键人物**: 尤利乌斯·凯撒 vs 维钦托利
- **关键战役**: 阿莱西亚之战 (公元前52) - 围攻战
- **地形**: 高原森林、木制堡垒
- **场景要点**: 罗马攻城器械、高卢木堡、双层围墙工事

## 中世纪 (公元500年 - 1500年)

### 【十字军东征】1095-1291年

- **交战方**: 欧洲十字军 vs 伊斯兰势力
- **关键地点**: 耶路撒冷、安条克、阿卡
- **兵种**:
  - 十字军: 重装骑士、十字弓手
  - 伊斯兰: 马穆鲁克骑兵、弓骑兵
- **场景要点**: 城堡攻城、重骑士冲锋、沙漠地形

### 【百年战争】1337-1453年

- **交战方**: 英格兰王国 vs 法兰西王国
- **关键战役**:
  - 克雷西战役 (1346) - 英格兰长弓手大胜
  - 阿金库尔战役 (1415) - 亨利五世的胜利
- **地形**: 法国平原、泥泞农田
- **兵种**:
  - 英格兰: 长弓手、重骑兵
  - 法兰西: 重骑士、弩手、火炮(后期)
- **场景要点**: 长弓齐射、重骑士冲锋、泥泞战场

### 【蒙古征服】1206-1324年

- **交战方**: 蒙古帝国 vs 各国
- **关键战役**:
  - 襄阳之战 (1267-1273) - 蒙古攻宋
  - 瓦尔施塔特战役 (1241) - 蒙古入侵欧洲
- **兵种**:
  - 蒙古: 骑射手、重骑兵、投石机、火器
- **场景要点**: 蒙古骑射战术、回回炮（配重投石机）

### 【怛罗斯之战】公元751年

- **交战方**: 唐朝安西军 vs 阿拉伯阿拔斯王朝
- **关键人物**: 高仙芝(唐) vs 齐亚德·本·萨利赫(阿拉伯)
- **兵力**: 唐军约2-3万 vs 阿拉伯军约5-7万
- **兵种**:
  - 唐军: 步兵、弩手、葛逻禄骑兵(战中叛变)
  - 阿拉伯: 骑兵、步兵
- **地形**: 中亚怛罗斯河流域(今哈萨克斯坦/吉尔吉斯斯坦边境)
- **历史意义**: 东西方文明在中亚的碰撞，造纸术西传的重要节点
- **场景要点**: 东西方军队对峙、骑兵战、葛逻禄骑兵倒戈场景

## 近代早期 (1500年 - 1800年)

### 【三十年战争】1618-1648年

- **交战方**: 神圣罗马帝国 vs 新教联盟 + 外国干涉
- **关键战役**: 布莱登菲尔德 (1631)、吕岑 (1632)-古斯塔夫二世阵亡
- **兵种**: 火枪兵、长矛兵、野战炮
- **场景要点**: 线列步兵、炮兵阵地、17世纪要塞

### 【大北方战争】1700-1721年

- **交战方**: 瑞典帝国 vs 俄罗斯/波兰/丹麦
- **关键人物**: 查理十二世 vs 彼得大帝
- **关键战役**: 波尔塔瓦战役 (1709)
- **兵种**: 滑膛枪兵、龙骑兵、野战炮
- **场景要点**: 瑞典战争机器、俄罗斯军事改革

## 近现代 (1800年至今)

### 【拿破仑战争】1803-1815年

- **交战方**: 法兰西第一帝国 vs 反法同盟
- **关键战役**:
  - 奥斯特里茨战役 (1805) - 三皇会战
  - 博罗季诺战役 (1812) - 法俄血战
  - 滑铁卢战役 (1815) - 拿破仑败亡
- **兵力**: 单战役兵力常达10万+
- **兵种**:
  - 法国: 老近卫军、胸甲骑兵、大军团
  - 反法同盟: 线列步兵、骑炮兵
- **地形**: 欧洲多样化地形
- **场景要点**: 大规模方阵、炮兵阵地、骑兵冲锋

### 【美国内战】1861-1865年

- **交战方**: 北方联邦军 vs 南方邦联军
- **关键战役**:
  - 葛底斯堡战役 (1863) - 转折点
  - 维克斯堡战役 (1863) - 攻城战
- **兵种**:
  - 步兵: 米尼弹步枪、线列战术
  - 火炮: 前装线膛炮
  - 骑兵: 侦察/游击
- **场景要点**: 19世纪野战工事、铁路运输、铁甲舰(海战)

### 【普法战争】1870-1871年

- **交战方**: 普鲁士王国 vs 法兰西第二帝国
- **关键战役**: 色当战役 (1870) - 拿破仑三世被俘
- **兵种**: 后装线膛枪(德莱赛步枪/夏塞波步枪)、钢制后装炮
- **场景要点**: 铁路快速集结、普鲁士总参谋部体系

### 【第一次世界大战】1914-1918年

- **交战方**: 协约国 vs 同盟国
- **关键战役**:
  - 索姆河战役 (1916) - 坦克首次使用
  - 凡尔登战役 (1916) - 战壕绞肉机
  - 加里波利战役 (1915) - 达达尼尔海峡
- **地形**: 战壕、无人区、弹坑
- **新式武器**: 坦克、飞机、毒气、机枪
- **场景要点**: 复杂战壕系统、早期坦克、毒气云、空中格斗

### 【第二次世界大战】1939-1945年

- **交战方**: 同盟国 vs 轴心国
- **关键战役**:
  - 斯大林格勒战役 (1942-43) - 城市巷战
  - 诺曼底登陆 (1944) - D-Day
  - 中途岛海战 (1942) - 航母对决
  - 突出部之役 (1944-45) - 装甲战
- **兵种**: 坦克、飞机、航母、步兵、火炮
- **场景要点**:
  - 斯大林格勒: 废墟城市战、狙击手
  - 诺曼底: 登陆艇、海滩碉堡、空降
  - 中途岛: 航母甲板、俯冲轰炸
  - 阿登森林: 虎式坦克、雪地森林

### 【朝鲜战争】1950-1953年

- **交战方**: 联合国军 vs 朝鲜/中国志愿军
- **关键战役**: 仁川登陆 (1950)、长津湖战役 (1950)
- **兵种**: 喷气式战机、坦克、机械化步兵
- **地形**: 朝鲜半岛山地
- **场景要点**: 喷气式空战(MiG-15 vs F-86)、山地坦克战

### 【海湾战争】1990-1991年

- **交战方**: 联军 vs 伊拉克
- **关键战役**: 沙漠风暴行动、73伊斯特ing战役 (装甲战)
- **兵种**: 主战坦克(M1A1/挑战者)、阿帕奇、隐形战机(F-117)
- **地形**: 沙漠
- **场景要点**: 夜视作战、GPS制导、沙漠地形、空中优势
